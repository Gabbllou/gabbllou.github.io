Exercício 1

Neste exercício, o programa irá pedir para o usuário digitar um número inteiro, e o programa irá identificar se
o número digitado é negativo, neutro ou positivo(estrutura condicional)

Exercício 2

Neste exercício, o programa irá pedir para o usuário digitar sua idade, a partir daí, o programa irá determinar se
você é criança, adolecente, adulto, ou idoso com base na sua idade(estrutura condicional)

Exercício 3

Neste exercício, o programa irá pedir para o usuário digitar um número de 1-7, a partir daí, o programa irá vincular este número a um dia da semana e irá mostrar para o usuário
exemplo: 1 = domingo(estrutura switch)

Exercício 4

Neste exercício, o programa irá automaticamente, por conta de um laço FOR, fazer uma sequência de 1 a 10, que vai aparecer na tela do usuário(laço de repetição "For")

Exercício 5

Neste exercício, o programa irá pedir para o usuário digitar números inteiros, até o usuário digitar o número 0, a partir dai, o programa irá somar todos os números automaticamente
e vai mostrar o resultado na tela(laços de repetição "Do While" e "For")

Exercício 6

Neste exercício, o programa irá colocar um número inteiro surpresa entre 1 e 10, e o programa irá pedir para o usuário adivinhar, se o usuário conseguir, vai receber um parabéns(Laço de repetição"Do While" e estrutura condicional)

Exercício 7

Neste exercício, o programa vai pedir para o usuário digitar números inteiros 5 vezes, a partir dai, o programa vai mostrar o menor e o maior número (Funções)

Exercício 8

Neste exercício, o programa vai pedir ao usuário uma quantidade de valores, daí o programa vai coletar esses valores, irá calcular a média, e vai exibr a média dos valores na tela(Funções, laço "For")

Exercício 9

Neste exercício, o programa vai pedir para o usuário digitar um número de 1 a 29, daí o programa vai comparar o número
contando quantos números faltam do escolhido pelo usuário até o número 29(estrutura condicional, laços de repetição)

Exercício 10

Neste exercício, o programa vai fornecer uma lista de palavras, e vai pedir para o usuário escolher uma digitando essa palavra
daí o programa vai mostrar a quantidade de vezes que essa palavra aparece no programa(Vetor, laço "For")
